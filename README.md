
# Run and deploy our CWV Auditor

View the app in AI Studio: https://ai.studio/apps/f5238f6b-b285-4c66-8ae6-1dfb4683e6f1

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
